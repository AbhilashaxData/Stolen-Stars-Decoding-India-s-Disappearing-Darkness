# Stolen-Stars-Decoding-India-s-Disappearing-Darkness
A 10-year radiometric study (2014-2023) of India’s night sky using NOAA VIIRS satellite data. Analyzing radiance trends across 7 regions from Hanle’s dark skies to Varanasi’s urban core to decode the "Jevon’s Paradox" and the loss of environmental capital. Includes Python scripts and the derived dataset.
